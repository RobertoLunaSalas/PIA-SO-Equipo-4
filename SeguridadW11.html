    Seguridad de Windows 11

    Las funciones de seguridad y privacidad de Windows 11 son similares a las de Windows 10. La seguridad de tus dispositivos comienza con el hardware e incluye la seguridad del sistema operativo, la seguridad de las aplicaciones y la seguridad del usuario y la identidad. Hay funciones disponibles en el sistema operativo Windows para ayudar en estas áreas. Esta sección describe algunas de estas características. Para una visión más completa, incluyendo la confianza cero.

    La aplicación Seguridad de Windows está integrada en el sistema operativo. Esta aplicación tiene una interfaz fácil de usar y combina las funciones de seguridad más utilizadas. Por ejemplo, tendrás acceso a protección contra virus y amenazas, cortafuegos y protección de red, protección de cuentas y mucho más. Las líneas base de seguridad** incluyen ajustes de seguridad ya configurados y listos para ser desplegados en sus dispositivos. Si no sabe por dónde empezar, o le lleva demasiado tiempo repasar todas las configuraciones, debería consultar las Líneas base de seguridad.

    Antivirus de Microsoft Defender** está integrado en Windows, y ayuda a proteger los dispositivos utilizando la seguridad de última generación. Cuando se utiliza con Microsoft Defender para punto de conexión, su organización obtiene una sólida protección de los puntos de conexión y una protección y respuesta avanzadas de los mismos. Si usa Intune para administrar dispositivos, puede crear directivas basadas en los niveles de amenaza en Microsoft Defender para punto de conexión. Las funciones de seguridad de las aplicaciones ayudan a evitar que se ejecute código no deseado o malicioso, aislar los sitios web y los archivos de Office que no son de confianza, proteger contra los sitios web de phishing o malware, y mucho más

    Windows Hello para empresas ayuda a proteger a los usuarios y las identidades. Reemplaza las contraseñas y utiliza un PIN o un dato biométrico que permanece localmente en el dispositivo. Los fabricantes de dispositivos están incluyendo características de hardware más seguras, como cámaras IR y chips TPM. Estas características se utilizan con Windows Hello para empresas para ayudar a proteger las identidades de los usuarios en los dispositivos de su organización. Como administrador, ir sin contraseña ayuda a asegurar las identidades de los usuarios. El sistema operativo Windows, Azure AD y Intune trabajan juntos para quitar contraseñas, crear directivas más seguras y ayudar a aplicar el cumplimiento.
